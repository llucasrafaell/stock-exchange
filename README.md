# 📈 Bolsa de valores

## Descrição do Projeto
Nesse projeto eu pego as informações das cotações de três moedas (Dólar, Euro e Bitcoin) utilizando uma API gratuita e apresento na tela utilizando a biblioteca tkinter no Python.

### 📌 Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
[Git](https://git-scm.com) e [Python](https://www.python.org/). 
Além disto é bom ter um editor para trabalhar com o código como [VSCode](https://code.visualstudio.com/) ou [PyCharm](https://www.jetbrains.com/pt-br/pycharm/download/).

Só é necessário instalar a biblioteca requets para utilizar a API.
#### 🎲 Instalar a biblioteca "Requests":
##### Abra o Prompt de Comando/Terminal Python e execute o comando:
```bash
$ pip install requests
```
#### 🎲 Link para importar a API 
```bash
$ https://economia.awesomeapi.com.br/last/USD-BRL,EUR-BRL,BTC-BRL
```
